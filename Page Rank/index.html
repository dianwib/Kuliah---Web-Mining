    2.3 Page Rank

    Page Rank¶

    Pengertian¶

    PageRank merupakan sebuah algoritma yang telah dipatenkan yang berfungsi untuk menentukan situs web mana yang lebih penting atau populer. PageRank merupakan salah satu fitur utama mesin pencari Google dan diciptakan oleh pendirinya, Larry Page dan Sergey Brin yang merupakan mahasiswa Ph.D. Universitas Stanford1.

    Algoritma Page Rank digunakan untuk pengambilan keputusan. Dalam project kali ini penulis akan membahas mengenai pencarian nilai masing-masing halaman yang tersedia pada sebuah website.

    Kita asumsikan ada sebuah website yang memliki 5 halaman, yaitu halaman A, B, C, D, E Masing-masing halaman memiliki satu atau beberapa tautan menuju halaman yang lain, untuk selanjutnya kita itung nilai page rank untuk masing-masing halaman. Data tautan yang tersedia adalah kurang lebih sebagai berikut;

    Halaman yang berisi tautan (sumber) Tujuan tautan pada halaman tersebut
    Halaman A Halaman B
    Halaman A Halaman D
    Halaman B Halaman E
    Halaman C Halaman A
    Halaman D Halaman E
    Halaman E Halaman A
    Halaman E Halaman C

    Code¶

    temp_rank=nx.pagerank(g,alpha=0.85,max_iter=500,tol=0.0001)

    Code diatas digunakan untuk menghitung nilai peringkat node atau web dalam sebuah graph yang diinisialisasi sebelumnya. PageRank merupakan algoritma yang dirancang untuk memberi peringkat halaman web.

Di dalam PageRank terdapat beberapa parameter sebagai berikut:
1. g (graph) - merupakan variabel tempat menampung graph yang diinisialisasi sebelumnya dalam format tipe data dictionary .
2. alpha (tipe float) - merupakan parameter yang menyatakan dumping untuk PageRank, secara default nilainya adalah 0.85.
3. max_iter (tipe integer) - merupakan parameter yang menyatakan jumlah maksimum iterasi dalam pemecah nilai eigenvelue algoritma PageRank.
4. tol (tipe float) - merupakan parameter yang menyatakan error toleransi kesalahan yang digunakan untuk memeriksa konvergensi dalam algoritma PageRank.

    from operator import itemgetter
a=1
for k,v in sorted(temp_rank.items(),key=itemgetter(1),reverse=True):
    print("ranking ke: ",a," adalah website >> ",k," : ",v)
    a+=1

    Code diatas digunakan untuk melakukan sorting nilai pagerank setiap website dimulai dari yang paling tinggi ke rendah.
