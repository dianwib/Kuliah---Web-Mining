    2.1 Alur Program

    Alur Program¶

    Pertama kali program akan mengunjungi website sumber yaitu https://www.liputan6.com/ , setelah itu program akan melakukan crawling link website yang terdapat pada website sumber (https://www.liputan6.com/), setelah menemukan link website pada website sumber maka program akan mengunjungi satu persatu link website yang telah ditemukan, begitu pula selanjutnya sampai program melakukan crawling pada website ke tiga (crawl 3 kedalaman).

    Selanjutnya program akan menggambar graph 2D dari hasil crawlink sebelumnya, dimana crawl sebelumnya disimpan dalam bentuk tipe data dictionary. Gambar akan berbentuk node atau hubungan antara website sumber dengan website tujuan.

    langkah 1 langkah 2 langkah 3
    crawl link pada website merdeka.com (kapanlagi.com, fimela.com,dll) crawl link pada website kapanlagi.com (liputan6.com, vemale.com, bola.com ,dll)
    crawl link pada website fimela*(kapanlagi.com, video,dll)*
    crawl link pada website liputan6.com (merdeka.com, bola.com, klikdokter.com ,dll) crawl link pada website bola.com (bola.net, brilio.net,dll) crawl link pada website bola.net, (fimela.com, bola.com ,dll)
    crawl link pada website brilio.net (liputan6.com ,dll)
    crawl link pada website merdeka.com (otosia.com, bintang.com ,dll) crawl link pada website otosia.com ( famous.id, bintang.com ,dll)
    crawl link pada website bintang.com *(liputan6.com, famous.id, fimela.com ,dll)

    kumpulan_link=[]
dict_link={}
