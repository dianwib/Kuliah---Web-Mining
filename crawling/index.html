    CRAWLING¶

    Pengertian¶

    Web crawler merupakan sebuah program yang memiliki prinsip kerja rekursif atau bekerja secara otomatis mengambil dan mengumpulkan semua informasi yang ada dalam suatu website. Informasi yang diambil tersebut dapat berupa sebuah text, gambar, video, maupun audio1.

    Prinsip kerja dari sebuah web crawler adalah akan mengunjungi setiap alamat website yang diberikan kepadanya, lalu program seakan-akan merayap pada website untuk melakukan filter tipe informasi yang akan diambil, kemudian web crawl mengambil dan menyimpan semua informasi berdasarkan filter yang terkandung didalam website tersebut, berdasarkan kategori. Setiap kali web crawler mengunjungi sebuah website, maka dia juga akan mendata semua link yang ada dihalaman yang dikunjunginya itu untuk kemudian dikunjungi lagi satu persatu, hasil informasi yang telah diambil tersebut biasanya disimpan di dalam sebuah database2.

    Pada bahasa pemrograman python, proses crawl informasi pada sebuah website dapat dilakukan dengan menggunakan library Beautifulsoup4.

    Tahapan¶

    Crawl link header seluruh berita pada website¶

    Langkah pertama yang dilakukan penulis adalah melakukan crawl / mengambil untuk mengambil seluruh url / link dari setiap berita yang ada didalam satu page website, hasil kumpulan link berita tersebut selanjutnya disimpan didalam file excel.

    tribunnews_url.py¶

    File tribunnews_url.py digunakan untuk mengambil setiap link berita pada setiap page url header, lalu disimpan pada file excel, proses pertama ketika melakukan crawl pada sebuah website berita adalah melakukan crawl / pengambilan setiap link berita pada header berita, kumpulan link yang sudah terambil tersebut selanjutnya akan kita kunjungi satu persatu untuk melakukan proses crawl judul dan isi setiap berita pada masing-masing link berita tersebut.
