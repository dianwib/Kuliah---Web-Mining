    TEXT PREPOCESSING¶

    Pengertian¶

    Text Preprocessing adalah tahapan dimana kita dapat melakukan proses filter data atau seleksi data agar data yang akan kita olah untuk tahap selanjutnya menjadi lebih terstruktur dan rapi. Pada umumnya tahapan proses text preprocessing terbagi menjadi 4 proses, yaitu ; Case Folding (konversi huruf kapital), Tokenisasi (pemotongan setiap kata pada dokumen), Stopword Removal (menghapus kata hubung), Stemming (pengambilan kata dasar dari kata yang memiliki imbuhan)1.

    Pada bahasa pemrograman python, proses text prepocessing dapat dilakukan dengan menggunakan library Sastrawi, library ini merupakan pengembangan dari library PHP Satrawi, dimana library tersebut dikhususkan untuk text berformat bahasa indonesia.

    Tahapan¶

    Tokenisasi¶

    Merupakan proses memecah dokumen text menjadi bagian-bagian kata, karena pada umumnya dokumen text sendiri terdiri dari banyak kalimat1. Proses ini dapat terjadi ketika kita menggunakan setiap fitur library Sastrawi. Di dalam proses ini juga terdapat proses Case Folding yaitu konversi huruf kapital serta penghapusan tanda baca seperti ; (titik, koma, dll).

    Stopword removal¶

    Stopword adalah kata hubung atau kata-kata hasil proses tokenisasi yang dianggap tidak deskriptif dan dapat dibuang seperti; "yang","di","dan", dll. Kata-kata tersebut pasti dapat kita temukan pada setiap dokumen dan memiliki frekuensi (tingkat kemunculan) banyak, penghapusan kata hubung tersebut tentunya dapat mengurangi ukuran index dan biaya komputasi serta mengurangi level noise2. Sedangkan Stopword removal merupakan proses menghapus kata hubung yang terdapat pada seluruh dokumen.

    tribunnews_sastrawi.py¶

    File tersebut melakukan semua proses prepocessing termasuk pengecekan setiap kata hasil proses tokenisasi pada database bahasa indonesia serta penambahan fitur stopword (penambahan untuk mengecek kata hubung) .

        factory = StopWordRemoverFactory()
    stopword = factory.create_stop_word_remover()
    stop = stopword.remove(kalimat)
    stop = re.sub(r'\b\w{1,2}\b', '', stop)

    code diatas digunakan untuk melakukan proses filtering / menghilangkan kata yang dianggap sebagai kata hubung pada dokumen menggunakan library default dari sastrawi StopWordRemoverFactory().

    Stemming¶

    Tujuan dari proses stemming adalah menstransformasikan kata menjadi kata dasarnya (root word) dengan menghilangkan semua imbuhan kata yang menempel2, namun ada beberapa kekurangan menurut penulis pada saat proses stemming menggunakan library Sastrawi, seperti kata "dimas" akan di transformasi menjadi kata "mas". Oleh karena itu diperlukan pengecekan setiap kata terhadap Kamus Besar Bahasa Indonesia ketika proses Text Prepocessing ini.

        factory = StemmerFactory()
    stemmer = factory.create_stemmer()
    katadasar = stemmer.stem(str(kalimat))
    katadasar = re.sub(r'\b\w{1,3}\b', '', katadasar).replace('-','')

    code diatas digunakan untuk melakukan proses stemming / proses menghilangkan imbuhan pada setiap kata yabf dianggap sebagai kata ber-imbuhan pada dokumen menggunakan library default dari sastrawi StemmerFactory().

    elif (tribunnews_cekkamus.cek_kata(kata)==True and tribunnews_cekstopword.cek_kata(kata)==True):

    code diatas digunakan untuk mengecek setiap kata apakah terdapat pada database kamus besar bahasa indonesia dan mengecek apakah kata tersebut termasuk stopword tambahan.

    tribunnews_cekkamus¶

    File tersebut melakukan pengecekan kata terhadap database kata KBBI (Kamus Besar Bahasa Indonesia).

    tribunnews_cekstopword¶

    File tersebut melakukan pengecekan kata hubung (stopword) tambahan yang terfilter, hal ini dikarenakan library stopword pada sastrawi kurang mengandung banyak kata hubung (stopword).

    *ketika file selesai di running maka akan terbentuk file excel baru dengan nama 'tribunnews_daftarsteaming.xlsx'

    *output file 'tribunnews_daftarsteaming.xlsx'
